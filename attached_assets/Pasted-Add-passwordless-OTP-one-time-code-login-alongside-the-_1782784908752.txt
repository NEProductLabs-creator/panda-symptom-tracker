Demo mode is gated behind /demo and uses a DemoProvider. The brand name is "PANS/PANDAS Companion".

Sign-in page UI. Add a third option labeled "Email me a sign-in code" below the Google and Apple buttons, separated by a divider. Sign-up page. Mirror the same OTP option on the sign-up page. New users created via OTP should hit the same onboarding flow as users who sign up with email + password.
Error handling. Handle these states explicitly with friendly copy aimed at stressed parents:

Invalid or expired code: "That code did not work. Try entering it again or send a new one."
Rate limited (Supabase returns 429): "Too many tries. Please wait a few minutes before requesting another code."
Network error: "We could not reach our servers. Check your connection and try again."
